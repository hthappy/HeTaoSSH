# 现代开发专用 SSH 工具 (代号：HetaoSSH) 需求文档

## 1. 项目概述

**定位**：一款轻量级、响应式、非商业化的开发者个人服务器管理利器。
**核心目标**：提供比传统终端更美观的 UI，比 VS Code 远程连接更轻量的启动速度，以及开箱即用的远程文件编辑能力。

## 2. 技术架构 (Technical Stack)

**框架**：Tauri 2.0 (跨平台，原生资源占用极低)。
**后端 (Rust)**：使用 `russh` 处理 SSH 通讯，`sqlx` + `SQLite` 存储配置。
**前端**：React / Vue 3 + Tailwind CSS + Shadcn/UI (现代化极简设计)。
**终端核心**：xterm.js (支持 WebGL 加速)。
**编辑器核心**：Monaco Editor (VS Code 同款内核)。


## 3. 功能需求 (Functional Requirements)


### 3.1 核心连接管理

**配置存储**：支持保存多个服务器配置（主机名、端口、用户名、认证方式）。
**认证支持**：支持 **密码登录*和 **私钥登录**（Ed25519, RSA 等）。
**多标签页 (Tab)**：支持同时开启多个 SSH 会话，且能快速切换。


### 3.2 现代化终端

**交互体验**：流畅的打字反馈，支持 24 位真彩色，兼容常用 CLI 工具（如 `htop`, `vim`）。
**快速搜索**：终端内全局文本搜索。
**一键复制粘贴**：支持点击式快速路径复制。



### 3.3 远程文件编辑 (核心亮点)

**双栏设计**：左侧文件树（SFTP 协议），右侧编辑/终端区。
**零感保存**：点击文件即开，`Ctrl+S` 直接将修改内容流式写回远程服务器。
**语法高亮**：对常用开发语言（Python, JS, Rust, Go, Config 等）提供完美高亮。


### 3.4 基础服务器管理

**资源概览**：侧边栏或顶部显示当前连接服务器的 CPU、内存、磁盘简单占用百分比（通过后台发送 `top` 或 `df` 命令解析）。
**命令片段 (Snippets)**：保存常用命令（如 `docker ps`, `nginx -s reload`），一键发送至终端。


## 4. 非功能性需求 (Non-functional Requirements)

**极致启动速度**：冷启动时间需控制在 **1.5秒*以内。
**低内存占用**：在维持单个 SSH 会话时，内存占用应低于 **80MB**。
**安全性**：本地存储的所有敏感信息（如服务器密码）必须经过 **AES-256*加密存储。
**可扩展性**：后端 Rust 层提供插件注册接口，前端 UI 预留扩展 Slot。


## 5. UI/UX 视觉规范



**整体风格**：玻璃拟态 (Glassmorphism) 或 极简暗黑风格。
**布局方案**：
**左侧栏**：服务器列表/文件树切换。
**中间主区**：采用 Tab 插件化布局，可以在“终端”与“编辑器”模式间无缝切换。
**底部状态栏**：显示延迟 (ms)、编码格式 (UTF-8) 和远程权限状态。


## 6. 后期扩展规划 (Future Roadmap)



1. **插件系统**：支持用户编写简单的脚本或加载 WASM 插件来扩展功能。
2. **多协议支持**：后期增加串口调试 (Serial Port) 和 Kubernetes Pod Exec 支持。
3. **同步功能**：支持通过私有的 Gist 或 WebDAV 备份加密后的服务器配置。
4. **智能联想**：结合本地历史记录，提供 SSH 终端内的智能命令提示。
